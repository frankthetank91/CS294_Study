```

- 스터디 형식: 매주 돌아가면서 그 주의 조가 준비한 자료를 참고해서 이론 공부 + 실습 + 질의응답 및 토론 
```
```
- 이론 공부 자료:
  
- 질의 응답: 강의 내용과 관련해 서로 질문하고 의견을 공유합니다.
```
